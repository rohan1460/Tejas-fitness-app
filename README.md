# 🏋️ Tejas – Smart Fitness Tracker

Tejas is a modern fitness web application designed to simplify and streamline personal health management. It enables users to efficiently track workouts, nutrition, and fitness goals with the power of AI from a single platform.

The goal of Tejas is to reduce manual effort, improve fitness consistency, and provide a structured approach to achieving health goals.

---

## 🚀 Features

- 🤖 AI Fitness Chatbot (Groq API) — personalized fitness advice
- 💪 Workout Tracking — log and monitor exercises
- 🍎 Nutrition Tracking — calorie and diet management
- 📊 Real-time Dashboard — live activity feed
- 🏆 Leaderboard — compete with others
- 🔔 Push Notifications — workout reminders
- 🌙 Dark Mode — eye-friendly interface
- 🔐 JWT Authentication — secure login system

---

## 🛠️ Tech Stack

- **Frontend:** React.js, CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Real-time:** Socket.io
- **AI:** Groq API
- **Auth:** JWT (JSON Web Tokens)
- **Version Control:** Git & GitHub
